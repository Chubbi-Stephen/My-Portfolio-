Intro

What exactly is your method to get the presence you deserve in other to thrive? A fully responsive and accessible web presence is no longer merely a 'bonus' or 'cool to show off'- it is ESSENTIAL!

It is of great privilege to offer the chance to be seen on the web to small businesses and individuals around me. Whether that be creating a website for you from the ground up, or updating an already existing presence for the mobile age. Creating a spot for parties from all over to find and learn about what you do is what i do. So how can i be of help today?

My Work

Here are a few of my projects, with description on the technologies used to achieve the final outcome, feel free to explore them.

Ceeman.

Designed and developed the RoyalCeemanglobal website for Ceeman Power Generator, a Nigerian power solutions brand founded in 2006. Built a responsive marketing and product showcase site highlighting the brand's core value propositions fuel efficiency, low noise operation, durability, and minimal maintenance. The site communicates Ceeman's mission of solving Nigeria's power challenges with generators engineered specifically for Africa's demanding conditions, presenting the brand's identity and product lineup in a clean, trustworthy interface suited for both B2C and B2B audiences.

Tranxact.

Designed and developed Tranxact, a full-stack fintech web application built with React and TypeScript for peer-to-peer money transfers with integrated AI financial assistance. Implemented core authentication flows including user registration and login with JWT token management stored in localStorage. Built a comprehensive dashboard featuring balance display, transaction history with filtering, and an interactive AI chat component for real-time financial advice. Developed a money transfer system using React Hook Form for form validation and error handling, with real-time transaction processing and success notifications.

Juddie Ecommerce.

Designed and developed Juddie Ecommerce, a dynamic online shopping platform powered by React, Redux, and API integration. Leveraged Tailwind CSS for a sleek, responsive, and visually engaging user interface. Seamlessly integrated backend APIs for robust functionality, enabling users to browse, search, and purchase products effortlessly. Implemented Redux for state management, ensuring a smooth and interactive shopping experience. Juddie Ecommerce represents a fusion of cutting-edge technology and seamless user experience, redefining online shopping for the modern era.

Official Website for the SixthSenseMusic Studio.

Transformed the digital presence of SixthSenseMusic Studio with a bespoke website crafted from scratch using HTML, CSS, JavaScript, and SCSS. Designed an immersive online platform that captures the essence of the studio's creativity and professionalism. Implemented intuitive navigation, visually appealing layouts, and seamless animations using JavaScript to create an engaging user experience. Leveraged SCSS for efficient styling, ensuring a responsive and visually cohesive design across all devices. The SixthSenseMusic Studio website stands as a testament to the harmonious blend of technical expertise and artistic vision, reflecting the studio's unique identity in the digital realm.

Moscath Garden Apartments.

Crafted a sleek, user-friendly website for Moscath, a premium short-term apartment rental company. Designed a seamless browsing experience, allowing guests to effortlessly explore and book meticulously curated apartments. Enhanced functionality, intuitive navigation, and ensuring a hassle-free experience for users. Elevate your stay with Moscath's exceptional shortlets, now just a click away.
